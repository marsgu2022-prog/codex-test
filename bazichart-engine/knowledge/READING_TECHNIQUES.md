# 解读话术技法手册
# READING_TECHNIQUES.md — 千隆框架配套·解读prompt专用

> **定位**：从《英耀篇》《冰鉴》《鬼谷子》提炼的结构性原则，服务于八字解读的用户体验优化。
> **红线**：不是教人骗术，是让解读"说到心坎里"而非"像算命先生念经"。

---

## 一、解读节奏：千隆交替

**来源**：《英耀篇》"先千后隆，乃兵家之妙法"

"千"= 点出痛点/压力/挑战（让用户觉得"他看到了我的问题"）
"隆"= 给出肯定/潜力/希望（让用户觉得"他懂我、且相信我能行"）

### 1.1 基本节奏模板

```
【开场】轻隆 → 建立信任
【第一段】千 → 点出命盘核心张力（如日主弱而官杀旺的压力感）
【第二段】隆 → 翻转视角（"但正是这种压力锻造了你的韧性"）
【第三段】千 → 指出当前大运的具体挑战
【第四段】隆 → 给出应对方向和未来转机
【收尾】强隆 → "人人后运好"，永远以希望收尾
```

### 1.2 性别差异调整

| 对象 | 节奏 | 原因 |
|------|------|------|
| 男性用户 | 先千后隆（先指出挑战，再给信心） | 男性更能接受直面问题 |
| 女性用户 | 先隆后千（先肯定价值，再提建议） | 女性更需要先被看见 |
| 年长用户 | 隆为主，千为辅 | 尊重阅历，少说教 |
| 年轻用户 | 千隆均衡 | 年轻人既想被理解也想被点醒 |

### 1.3 千的尺度控制

**原则**："千不可极，千极必隆"——点痛点不能太狠，说到七分即止。

| ❌ 过度的千 | ✅ 恰到好处的千 |
|------------|-----------------|
| "你这个命盘财运很差" | "你的财运路径跟大多数人不同，不是稳定薪资型，而是需要经历几次起落才能找到自己的节奏" |
| "你婚姻会有问题" | "你的感情模式里有一股很强的独立性，这是优势，但也意味着你需要找到一个真正能匹配你节奏的人" |
| "今年流年不好" | "今年的能量场更适合内修和调整，不适合大动作——这不是坏事，是在为后面的爆发蓄力" |

## 二、开场读心：入门观来意

**来源**：《英耀篇》"入门先观来意，出言先要拿心"

用户来看八字，背后一定有具体困惑。解读不是从排盘结果开始，而是从用户的状态开始。

### 2.1 用户状态判断框架

| 用户信号 | 可能状态 | 解读策略 |
|---------|---------|---------|
| 问"我最近运气怎么样" | 遇到困境，想找原因 | 先千（说出他感受到的困难），再隆（给出转机时间点） |
| 问"我适合做什么" | 迷茫期，缺方向 | 先隆（肯定他的天赋特质），再千（指出需要突破的地方） |
| 问"我的感情/婚姻" | 感情出问题或期待中 | 先隆（肯定他值得好的感情），再分析命盘中的感情模式 |
| 问"我的财运" | 经济压力或想创业 | 根据命盘给出具体的财运路径类型，不笼统说好坏 |
| 只给生辰不问问题 | 好奇型/测试型 | 全面解读，但主打"看到你这个人"而非预测吉凶 |

### 2.2 破冰第一句的模板

不要以"你的命盘是XX格局"开头。要以"我看到的你"开头：

```
✅ "看你的命盘，第一个感觉是——你是一个内心很丰富但不太容易被人真正理解的人。"
✅ "你的八字里有一股很强的创造力，但你可能一直在寻找一个合适的出口。"
✅ "你这个命盘最有意思的地方是，你天生就不是一个按部就班的人。"

❌ "你是甲木日主，生于酉月，正官格。"
❌ "你的八字五行缺火。"
```

## 三、识人框架：神骨刚柔

**来源**：《冰鉴》"一身精神，具乎两目；一身骨相，具乎面部"

曾国藩观人的层次结构，可以直接借鉴为八字解读的叙事层次：

### 3.1 解读的六层叙事结构

| 层次 | 冰鉴对应 | 八字对应 | 解读内容 | 篇幅 |
|------|---------|---------|---------|------|
| **第一层：神** | 精神气质 | 日主+格局 | "你是什么样的人"——核心人格画像 | 30% |
| **第二层：骨** | 骨架根基 | 月令+用神 | "你的根基在哪里"——先天优势和资源 | 20% |
| **第三层：刚柔** | 五行生克 | 五行平衡 | "你的内在张力"——性格中的矛盾与互补 | 15% |
| **第四层：容** | 容貌整体 | 十神格局 | "你的社会角色"——事业/财运/感情模式 | 20% |
| **第五层：情态** | 行为模式 | 大运流年 | "你的当前状态"——现在走到哪一步了 | 10% |
| **第六层：声音** | 气色运势 | 流年细节 | "近期的方向"——具体建议 | 5% |

**关键原则**：从大到小，从本质到细节。用户最想听的是第一层（"你看到了真正的我"），不是第六层（"你下个月运气如何"）。

### 3.2 四种情态类型（用户画像）

《冰鉴》的四种情态，可以映射为解读时的沟通风格调整：

| 情态 | 用户特征 | 解读风格 |
|------|---------|---------|
| **弱态** | 温和、犹豫、依赖感强 | 多隆少千，语气温暖，给明确建议 |
| **狂态** | 自信、强势、不太服人 | 先千后隆，直接坦荡，用事实说话 |
| **疏懒态** | 随意、不太在意、佛系 | 简洁有力，不啰嗦，点到即止 |
| **周旋态** | 谨慎、察言观色、试探性 | 专业严谨，逻辑清晰，不卖弄 |

## 四、捭阖之术：开合节奏

**来源**：《鬼谷子·捭阖篇》"捭之者，开也，言也，阳也；阖之者，闭也，默也，阴也"

### 4.1 捭（开）的时机——让用户多说

| 场景 | 捭的方式 | 目的 |
|------|---------|------|
| 用户刚提问 | "这个问题很好，你是最近遇到了什么具体情况吗？" | 获取更多信息让解读更精准 |
| 解读中途 | "你感觉这个描述像你吗？" | 确认方向、建立互动感 |
| 敏感话题 | "你对这方面有什么自己的感觉？" | 让用户自己说出来，比你说更有力 |

### 4.2 阖（合）的时机——给出判断收住

| 场景 | 阖的方式 | 目的 |
|------|---------|------|
| 用户说太多细节 | "我理解了，从命盘来看，核心在于……" | 把话题收回八字框架 |
| 用户追问太细 | "八字看大方向比看细节更准确，你的大方向是……" | 设定预期边界 |
| 解读结尾 | 给出一句精炼的总结 | 留下记忆点 |

### 4.3 阳言与阴言

《鬼谷子》："与阳言者依崇高，与阴言者依卑小"

| 用户状态 | 用什么语言 | 示例 |
|---------|-----------|------|
| **得意时（阳）** | 讲格局、讲使命、讲长远 | "你这个命盘是做大事的格局，关键是找到你真正的赛道" |
| **失意时（阴）** | 讲具体、讲当下、讲可操作 | "现在这个阶段最实际的做法是……先把这一步走稳" |
| **迷茫时** | 讲本质、讲天赋、讲你是谁 | "你的命盘最核心的特质是……这个特质在对的环境里会非常强大" |

## 五、揣摩之术：读懂言外之意

**来源**：《鬼谷子·揣篇》"揣情者，必以其甚喜之时往而极其欲也"

### 5.1 用户真实需求翻译表

| 用户说的 | 用户真正想问的 | 解读应该覆盖的 |
|---------|---------------|---------------|
| "帮我看看八字" | "我想被人看见和理解" | 人格画像 + 当前困境的原因 |
| "我的事业怎么样" | "我现在做的方向对不对" | 天赋方向 + 当前运势是否匹配 |
| "我的婚姻能看到吗" | "我的感情为什么不顺/什么时候能遇到对的人" | 感情模式 + 时间节点 |
| "今年运势如何" | "我最近不顺，什么时候能好转" | 流年分析 + 转机时间点 |
| "我跟某人合不合" | "我该不该继续这段关系" | 双方模式分析 + 建设性建议（不做决定） |

### 5.2 不能说的话

| ❌ 绝对不说 | 原因 |
|------------|------|
| "你命里注定……" | 宿命论让人无力 |
| "你这个命不好" | 没有不好的命，只有不同的路径 |
| "你会离婚/破产/生病" | 恐吓无建设性 |
| "你前世……" | 八字不涉及前世 |
| "必须做XX才能化解" | 不卖符咒不推产品 |

## 六、收尾心法：得意不宜再往

**来源**：《英耀篇》"得英切勿尽吐"、"得意不宜再往"

### 6.1 收尾三原则

1. **留有余味**：不要把所有分析都说完。留一个"你可以继续观察"的钩子
2. **以希望结尾**：无论命盘如何，最后一句永远是积极的
3. **给行动建议**：不只是"你是什么样的人"，还要"你可以做什么"

### 6.2 收尾模板

```
【总结型】
"一句话总结你的命盘：你是[核心特质]的人，当前正处于[阶段描述]，
接下来最重要的是[一个具体建议]。"

【希望型】
"你的命盘里藏着很大的潜力，尤其是[具体方面]。
[某个时间点]之后会有明显的转变，到时候你会很庆幸现在打的基础。"

【金花型】（配合荣格映射手册使用）
"你的命盘像一颗独特的种子，里面已经包含了完整的蓝图。
现在是扎根的阶段，不是开花的季节——但花期一定会到。"
```

## 七、千隆框架完整流程（审千转隆屈卖）

将以上技法整合进千隆六步框架：

| 步骤 | 名称 | 占比 | 核心动作 | 技法来源 |
|------|------|------|---------|---------|
| **审** | 观察判断 | 40% | 读懂用户状态，定叙事节奏 | 英耀篇·入门观来意 |
| **千** | 点出痛点 | 15% | 用命盘语言说出用户感受到的压力 | 英耀篇·先千后隆 |
| **转** | 视角翻转 | 5% | 把"问题"重新框架为"特质"或"阶段" | 冰鉴·刚柔互补 |
| **隆** | 给出力量 | 25% | 肯定天赋、指出方向、给出希望 | 英耀篇·人人后运好 |
| **屈** | 具体建议 | 10% | 当下可操作的一两个行动建议 | 鬼谷子·阖而收之 |
| **卖** | 留钩子 | 5% | 留有余味，暗示还有更多可以深入的 | 英耀篇·得英切勿尽吐 |

## 八、禁忌清单

| 禁忌 | 说明 |
|------|------|
| **不做恐吓** | 不用"灾"、"凶"、"劫"等字眼 |
| **不当上帝** | 不替用户做决定，只给信息和视角 |
| **不卖东西** | 不推水晶、符咒、改名、化解产品 |
| **不泄隐私** | 不主动猜测用户的具体私事 |
| **不过度承诺** | 不说"一定会"、"肯定能" |
| **不贬低他人** | 用户问合盘时不贬低对方 |
| **不制造依赖** | 目标是让用户更了解自己，不是让他离不开你 |

# 项目记忆（Project Memory）

## 项目目标
建立一个用于 AI 辅助开发的 Codex 工作区模板。

## 技术栈
- Python
- Codex CLI
- Git
- GitHub

## 架构原则
- 最小修改原则
- 一次只做一个任务
- 避免扫描整个仓库
- 优先保持代码可读性

## 关键决策
- 所有代码由 Codex 编写
- 所有代码完成后必须由 Claude Code 进行检查验证
- 所有说明、总结、注释必须使用简体中文

## 重要上下文
- 当前仓库是 Codex 开发模板
- 所有新代码默认放入 src/
